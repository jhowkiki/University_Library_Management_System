University_Library_Management_System
The University Library Management System aims to control loans, returns, reading history, and student/reader ratings based on their use of the university library.

Project Scope:
The function of the University Library Management System is to register students and books, record the borrowing and return of books with date and time, classify students/readers according to the number of books read in the semester, and consult available books, loan history, and general reader classification.
